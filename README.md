Train Departures!

This is a simple site for phone / tablet / home displays that can show the upcoming departures for the chosen station.

Its been made to resemble the old style depature boards on the UK rail network, to a certain extent.

This is possible thanks to https://huxley2.azurewebsites.net/ which provides an easy to use access point to the LDB API from browser-based client-side PWAs made with JavaScript or TypeScript. Additionally, it opens up the Windows enterprise API to agile developers on macOS and Linux.
